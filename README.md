# Safe Harbor Tech Portal

Network operations dashboard for managing UniFi and Meraki sites.

## Features
- 34+ sites monitored (UniFi Cloud + Meraki)
- Real-time device and client statistics
- Network health monitoring
- Service ticket system
- Wireless optimization recommendations

## Deployment

### Streamlit Cloud
1. Push to GitHub
2. Connect to share.streamlit.io
3. Add secrets in dashboard:
   - `UNIFI_API_KEY`
   - `MERAKI_API_KEY`

### Local Development
```bash
pip install -r requirements.txt
streamlit run app.py
```
